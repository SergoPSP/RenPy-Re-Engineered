# RenPy-Re-Engineered
Ren'Py: Re-Engineered — это высокооптимизированная переработка движка Ren'Py. Парсинг rpy-скриптов и выполнение Python-кода полностью переписаны на C++ и MicroPython, что дает до 20x прирост скорости запуска и 10-кратное снижение потребления ОЗУ. Полная обратная совместимость с существующими играми и поддержка мобильных платформ (PSP, Vita, 3DS).
